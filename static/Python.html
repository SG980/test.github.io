1.2 标识符命名规定

介绍：标识符指的是Python中的变量名、函数名、类名等一系列的名字，Python对这些名字的命名给予了一些规定，必须遵守。

 

标识符命名规则：

  1. 只能由数字、字母、_（下划线）组成

  4. 严格区分大小写

5. 格式化输出

介绍：格式化输出的含义就是按照一定的格式输出内容，目前Python有三种格式化输出的方式，掌握其中一种即可，推荐大家掌握 f表达式。

5.1 f表达式

介绍：在字符串前写上一个 f 或 F 即可该字符串开启了格式化。在字符串的花括号中可以写入表达式。

 

示例一：{}中调用变量

x
1
name = 'tuoni'
2
age = 18
3
info = f'名字:{name}  年龄:{age}'
4
print(info) #  名字:tuoni  年龄:18

示例二：{}中做运算

x
1
print(f'计算1+1的和：{1+1}')

示例三：保留小数

1
info = 10.123
2
print(f'保留两位小数后打印：{info:.2f}')  # 10.12
3
print(f'保留一位小数后打印：{info:.1f}')  # 10.1

5.2 %格式化

介绍：%的主要作用将数据转换为指定的输出格式。就是占位符，主要是通过%的方式，将数字、字符传递到字符串里所在位置，传递的时候按照顺序传。

 

字符类型：

注意：一般只要记住 %s 和 %d 就好了，这两个占位符的使用频率最高

 

示例：

x
9
1
info = 'hello'
2
number = 10
3
print("%s" % info)  # 转换多少个，字符就需要输入多少个参数
4
print("%s%d" % (info, number))  # 打印hello10
5
print("%s%d%s" % (info, number, info))  # 打印hello10hello
